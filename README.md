Linkresshd
------
使用 Go 语言编写的 ssh 服务端程序，可以只通过一个二进制文件运行来实现 ssh 服务，一个小型轻量级的 ssh 服务只需要一个文件即可完成，并且还拥有高度的可自定义化，方便不同场景使用。

<br />

使用方法
------
下载发行版的二进制程序或者自行编译，可使用 go build 编译。

再直接运行二进制文件即可。

<br />

配置文件
------
在 `config/config.json` 中 (此文件必须存在)，您可以自行修改。
```
"name": "root",         登录的用户名
"password": "toor",     登录的密码
"command": "/bin/bash", 登录之后运行的命令
"port": 2222            监听的端口
```
